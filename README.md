# 📊 HR Attrition Analysis

🚀 **End-to-End Data Analysis Project**
End-to-end data analysis project leveraging Python, Pandas, SQL, and Excel to derive actionable insights from the IBM HR Analytics dataset. Investigated attrition rate, departmental differences, overtime impact, salary trends across roles, and age-group-based attrition patterns.


## 📌 Project Overview

Employee attrition is a critical challenge for organizations. This project explores key factors influencing attrition using the **IBM HR Analytics Employee Attrition & Performance dataset**.

The goal is to identify trends and patterns that can help businesses reduce employee turnover and improve retention strategies.


## 🎯 Key Business Questions Answered

* 📉 What is the overall attrition rate?
* 🏢 Which department has the highest attrition?
* ⏱️ Does overtime affect attrition?
* 💰 What is the average monthly income by job role?
* 🎂 Which age group has the highest attrition?


## 🛠️ Tools & Technologies Used

* **Python (Pandas)** → Data cleaning & analysis
* **SQL** → Query-based insights
* **Microsoft Excel** → Data exploration & validation

## 📊 Key Insights

### 🔹 1. Overall Attrition Rate

* A significant portion of employees leave the company, indicating potential organizational or cultural issues.


### 🔹 2. Department-wise Attrition

* Certain departments show **higher attrition**, highlighting workload imbalance or job dissatisfaction.


### 🔹 3. Impact of Overtime

* Employees working overtime are **more likely to leave**, suggesting burnout as a major factor.


### 🔹 4. Salary Trends by Job Role

* Lower-paying roles tend to have **higher attrition**, indicating compensation plays a key role in retention.


### 🔹 5. Age Group Analysis

* Younger employees show **higher attrition**, possibly due to career switching or growth opportunities.

This analysis highlights that **overtime, salary, department, and age** are key drivers of employee attrition. Organizations can use these insights to:

* Improve work-life balance
* Optimize compensation strategies
* Focus on high-risk departments
* Retain younger talent
