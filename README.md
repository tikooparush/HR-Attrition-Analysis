# HR-Attrition-Analysis
End-to-end data analysis project leveraging Python, Pandas, SQL, and Excel to derive actionable insights from the IBM HR Analytics dataset. Investigated attrition rate, departmental differences, overtime impact, salary trends across roles, and age-group-based attrition patterns.
